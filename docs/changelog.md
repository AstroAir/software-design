# 更新日志

本文档记录项目的版本更新历史。

## [1.0.0] - 2024-12-03

### 新增

- 完整的校园卡管理功能
  - 开卡、充值、挂失、解冻
  - 密码管理和重置
- 上机计费系统
  - 自动计时计费
  - 按分钟精确结算
- 双角色认证
  - 学生登录（卡号+密码）
  - 管理员登录（密码）
- 统计报表
  - 每日收入统计
  - 上机记录查询
- 数据管理
  - JSON 文件存储
  - 数据导入导出
  - 模拟数据生成
- 现代化 UI
  - Fluent Design 风格
  - ElaWidgetTools 组件库

### 技术特性

- C++20 标准
- Qt6 框架
- CMake Presets 构建配置
- 跨平台支持（Windows/Linux/macOS）

---

## 版本规范

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

版本号格式：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 变更
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修复
