# 使用教程

本教程将引导你了解校园卡管理系统的基本操作流程。

## 默认账户

系统首次运行时会自动创建以下测试账户：

| 角色 | 凭据 | 说明 |
|------|------|------|
| **管理员** | 密码：`admin123` | 系统管理员账户 |
| **学生 1** | 卡号：`C001`，密码：`123456` | 张三 |
| **学生 2** | 卡号：`C002`，密码：`123456` | 李四 |
| **学生 3** | 卡号：`C003`，密码：`123456` | 王五 |

## 学生操作

### 登录系统

1. 启动程序，在登录界面选择「**学生**」角色
2. 输入卡号（如 `C001`）
3. 输入密码（默认 `123456`）
4. 点击「**登录**」按钮

!!! tip "提示"
    密码连续错误 3 次后账户将被冻结，需联系管理员解冻。

### 开始上机

1. 登录成功后进入学生面板
2. 点击「**开始上机**」按钮
3. 在弹出的对话框中选择上机地点
4. 确认后开始计时计费

### 结束上机

1. 点击「**结束上机**」按钮
2. 系统自动计算上机时长和费用
3. 从余额中扣除费用
4. 显示本次上机详情

### 查看记录

在学生面板可以查看：

- **上机记录** - 历史上机明细
- **统计信息** - 总上机次数、总时长、总费用
- **筛选功能** - 按日期范围和地点筛选记录

### 修改密码

1. 点击「**修改密码**」按钮
2. 输入当前密码
3. 输入新密码并确认
4. 点击确定完成修改

## 管理员操作

### 登录系统

1. 在登录界面选择「**管理员**」角色
2. 输入管理员密码（默认 `admin123`）
3. 点击「**登录**」按钮

### 卡片管理

#### 添加新卡

1. 点击「**添加新卡**」按钮
2. 填写卡号（或点击自动生成）
3. 填写姓名、学号
4. 设置初始密码
5. 点击确定创建

#### 充值

1. 在卡列表中选择目标卡
2. 点击「**充值**」按钮
3. 输入充值金额
4. 确认充值

#### 挂失/解挂

- **挂失**：选中卡后点击「挂失」，卡将无法登录
- **解挂**：选中已挂失的卡，点击「解挂」恢复正常

#### 解冻

当学生密码错误次数过多导致冻结时：

1. 选中被冻结的卡
2. 点击「**解冻**」按钮
3. 卡状态恢复正常，错误次数清零

#### 重置密码

1. 选中目标卡
2. 点击「**重置密码**」
3. 输入新密码
4. 确认重置

### 统计报表

1. 点击「**统计**」按钮
2. 选择查询日期
3. 查看当日收入、上机次数、总时长
4. 查看详细记录列表

### 数据管理

#### 导出数据

1. 点击「**导出**」按钮
2. 选择保存位置
3. 数据将导出为 JSON 文件

#### 导入数据

1. 点击「**导入**」按钮
2. 选择要导入的 JSON 文件
3. 选择合并或覆盖现有数据

#### 生成模拟数据

用于测试目的：

1. 点击「**生成模拟数据**」
2. 设置卡数量和每卡记录数
3. 系统自动生成测试数据

### 修改管理员密码

1. 点击「**修改管理员密码**」
2. 输入当前密码
3. 输入新密码并确认
4. 点击确定完成修改

## 计费规则

| 项目 | 说明 |
|------|------|
| **费率** | 1.0 元/小时 |
| **计费方式** | 按分钟计费，不足一分钟按一分钟计算 |
| **扣费时机** | 结束上机时从余额扣除 |

!!! example "计费示例"
    上机 1 小时 30 分钟 = 90 分钟 × (1.0 ÷ 60) = 1.5 元

## 卡状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| **正常** | 🟢 | 可正常登录和上机 |
| **挂失** | 🟡 | 无法登录，需管理员解挂 |
| **冻结** | 🔴 | 密码错误过多，需管理员解冻 |
